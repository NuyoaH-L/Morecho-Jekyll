## 开始

1. 克隆或者fork这个仓库
1. 在你的电脑安装ruby
1. 运行命令

```bash
$ gem install bundler jekyll
$ cd /path/to/this/repo
$ bundle install
$ bundle exec jekyll serve
```

你将会在这看到你的博客 [here](http://localhost:4000)
